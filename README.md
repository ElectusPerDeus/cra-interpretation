# @platform/cra-interpretation

Deterministic CRA interpretation package extracted for standalone consumption.

## Included modules

- evaluator
- contracts
- manifest types
- coverage map
- manifest hash
- service taxonomy and labels
- version constants
- manifest and schema assets

## Scripts

- `npm test`
- `npm run typecheck`
- `npm run lint`

## Install by exact SHA (example)

```bash
npm install github:ElectusPerDeus/cra-interpretation#<commit-sha>
```
